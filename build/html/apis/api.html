API¶

pytsk.api¶

class pytsk.torch_model.TSK(in_dim, out_dim, n_rules, order=1, antecedent='tsk')¶

Parent: torch.nn.Module.

TSK model.

Parameters

  • in_dim (int) – Input dimension \(D\).

  • out_dim (int) – Output dimension, \(C\) for C-class classification problems, \(1\) for single output regression problems.

  • n_rules (int) – Number of rules \(R\) of the TSK model.

  • order (int) – 0 or 1. default=1. If order=1, first-order TSK is generated, else zero-order TSK is generated.

  • antecedent (str) – default="tsk". Define the antecedent part of the TSK model. Support: tsk, htst, logtsk, or custom Antecedent() class.

init_model(self, X, y=None, scale=1., std=0.2, method='cluster', sigma=None, cluster_kwargs=None, eps=1e-8)¶

The antecedent parameters are initialized by this. Must be called before training.

Parameters

  • X (numpy.array) – Numpy array input with size \((N, D)\) for antecedent parameters initialization.

  • y (numpy.array) – Numpy array input with size \((N)\) for antecedent parameters initialization. This parameter is reserved for custom Antecedent modules or further version, and is not used in this version.

  • scale (float) – Parameter for initializing the standard deviation \(\sigma\) of Gaussian fuzzy sets in each rule, \(\sigma\sim\mathcal{N}(\text{scale}, \text{std})\).

  • std (float) – Parameter for initializing the standard deviation \(\sigma\) of Gaussian fuzzy sets in each rule, \(\sigma\sim\mathcal{N}(\text{scale}, \text{std})\).

  • method (str) – Method for initializing the center of antecedent parameters, support: random, cluster. If :: random: The center of the rules are selected randomly from X.
